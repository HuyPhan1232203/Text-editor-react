# Text-Editor-React

Rich text editor built with React and TipTap for document generation.

## Features

- Rich text formatting (bold, italic, etc.)
- Tables, lists, images
- DOCX export

## Quick Setup

1. Clone: `git clone <repo-url>`
2. Install: `pnpm install`
3. Run: `pnpm run dev`

Open `http://localhost:5173` to edit and export documents.
